# Asclépio

## Tipo

Projeto pessoal.

## Objetivo

Documentar um aplicativo de saúde voltado ao acompanhamento do bem-estar do usuário, com recursos para hidratação, monitoramento de peso, registro de exercícios, histórico de sintomas e alertas de surtos de doenças na região.

## Tecnologias

- Flutter
- Firebase
- Android Studio
- Shared Preferences
- Integração com GPS
- Git e GitHub
