# AMET

## Tipo

Projeto pessoal.

## Objetivo

Documentar um aplicativo inspirado na experiência do Rave, voltado para pessoas assistirem filmes e vídeos de forma sincronizada em salas compartilhadas.

## Tecnologias

- Reprodução de vídeos
- Salas compartilhadas
- Sincronização de sessão
- Desenvolvimento de aplicativo
- Git e GitHub

## Organização
