# PortfolioHUB

## Tipo

Projeto acadêmico.

## Objetivo

Criar um repositório público com organização profissional, documentação adequada, histórico de versionamento, tag de entrega e publicação por GitHub Pages.

## Tecnologias

- HTML
- CSS
- JavaScript
- Git
- GitHub
- GitHub Pages

## Organização

O projeto é composto por uma página inicial em `index.html`, estilos em `assets/css/styles.css`, interações em `assets/js/main.js`, relatório em HTML, apresentação em HTML e documentação distribuída por READMEs.

## Status

Versão inicial concluída para entrega intermediária.

## Próximos passos

- Adicionar prints da publicação e do histórico de commits ao relatório final.
- Usar o repositório como base para novos projetos acadêmicos e profissionais.
