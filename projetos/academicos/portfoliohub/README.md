# PortfolioHUB

Projeto principal da entrega final. Centraliza portfólio, documentação, evidências, relatório em PDF e slides de apresentação.

## Tecnologias

- HTML
- CSS
- JavaScript
- GitHub REST API
- GitHub Pages

## Pontos avaliáveis

- Implantação documentada.
- Integração com GitHub.
- Controle de acesso e segurança.
- Testes.
- Apresentação final.
