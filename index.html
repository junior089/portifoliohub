Sobre

Um hub para projetos, documentação e presença profissional

O PortfolioHUB reúne projetos acadêmicos, projetos pessoais, documentação técnica e uma página pública preparada para GitHub Pages. A proposta é apresentar a evolução em desenvolvimento web, organização de repositórios e boas práticas de versionamento.

01

Organização

Pastas separadas por documentos, slides, assets e projetos acadêmicos ou pessoais.

02

Versionamento

Histórico Git com commits por etapa, branch main e tag v1.0 para a primeira versão.

03

Entrega pública

Página publicada, relatório em PDF e links profissionais conectados ao GitHub e LinkedIn.
