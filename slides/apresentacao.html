PortfolioHUB

Repositório acadêmico e profissional com projetos documentados, versionamento Git/GitHub e publicação via GitHub Pages.

Roteiro de falaApresentar o nome do projeto, o contexto da entrega intermediária e a proposta de criar um hub público para organizar projetos, documentação e evolução profissional.
Slide 02

Objetivo do repositório

  • Reunir projetos acadêmicos e pessoais em uma estrutura única.
  • Demonstrar boas práticas de documentação e versionamento.
  • Preparar uma página pública para avaliação e portfólio.
Roteiro de falaExplicar que o repositório não é apenas uma pasta de arquivos, mas uma apresentação organizada do aprendizado, com histórico e documentação acessíveis.
Slide 03

Estrutura de pastas

  • assets/ concentra CSS, JavaScript e imagens.
  • documentos/ contém o relatório final em HTML.
  • slides/ contém esta apresentação.
  • projetos/ organiza projetos acadêmicos e pessoais.
Roteiro de falaDestacar que a estrutura facilita navegação, avaliação e manutenção futura, mantendo no repositório apenas materiais públicos e finais.
Slide 04

Projeto acadêmico principal

O PortfolioHUB é o projeto central da entrega intermediária, construído com HTML, CSS e JavaScript puro.

  • Página inicial responsiva.
  • Documentação em README.
  • Relatório final preparado para PDF.
Roteiro de falaComentar que a escolha por tecnologias simples garante funcionamento direto no GitHub Pages e facilita a avaliação do conteúdo.

Projetos pessoais documentados
