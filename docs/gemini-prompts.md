# Uso do Google Gemini

O Google Gemini foi usado como apoio durante a implantação, sem expor chave de API no código-fonte.

## Prompt 1 - Plano de implantação

> Crie um plano de implantação para uma plataforma chamada PortfolioHUB, com GitHub, segurança, controle de acesso, testes e apresentação final.

Resultado aplicado: criação do cronograma da entrega e divisão do projeto em etapas.

## Prompt 2 - Segurança

> Quais práticas de segurança devo documentar em um projeto acadêmico hospedado no GitHub Pages?

Resultado aplicado: CSP, matriz IAM, branch protection, secret scanning e ausência de credenciais.

## Prompt 3 - Testes

> Crie um checklist de testes para validar uma aplicação estática de portfólio integrada com GitHub API.

Resultado aplicado: testes de navegação, responsividade, fallback da API, links e relatório final.

## Prompt 4 - Apresentação

> Monte um roteiro de vídeo de até cinco minutos para apresentar a implantação do PortfolioHUB com GitHub e Gemini.

Resultado aplicado: criação dos slides HTML e do roteiro em `docs/video-roteiro.md`.
