# PortfolioHUB

PortfolioHUB é um repositório acadêmico e profissional criado para organizar projetos, documentação e materiais de apresentação em uma estrutura pública, limpa e versionada com Git/GitHub.

O projeto foi desenvolvido por Carlos Alberto Junior como parte do desafio **Entrega Intermediária: Criação de Repositório com Versionamento**.

## Objetivo acadêmico

Demonstrar a criação de um repositório público com boa organização de arquivos, documentação clara, histórico de commits, tag de versão e publicação por GitHub Pages.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Git
- GitHub
- GitHub Pages

## Como visualizar localmente

Abra o arquivo `index.html` diretamente no navegador. O projeto não depende de instalação, build ou bibliotecas externas obrigatórias.

## Como publicar no GitHub Pages

1. Acesse o repositório no GitHub.
2. Entre em **Settings > Pages**.
3. Em **Build and deployment**, selecione **Deploy from a branch**.
4. Escolha a branch `main` e a pasta `/root`.
5. Salve a configuração e aguarde a publicação.

## Estratégia de versionamento

O histórico do projeto foi organizado em commits incrementais, separando estrutura inicial, página principal, documentação, apresentação e preparação para publicação.

## Histórico planejado de commits

- `chore: estrutura inicial do PortfolioHUB`
- `feat: adiciona página principal do portfólio`
- `docs: adiciona documentação dos projetos e relatório final`
- `docs: adiciona apresentação em HTML para vídeo final`
- `chore: prepara publicação no GitHub Pages`
